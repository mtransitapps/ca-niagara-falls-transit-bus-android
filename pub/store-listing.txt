Full description:
This app adds Niagara Falls Transit buses information to MonTransit.

This app provides the buses schedule (available offline).

Niagara Falls Transit buses serve Niagara Falls in Ontario, Canada.

Once this application is installed, the MonTransit app will display buses information (schedule...).

This application only has a temporary icon and no screen: download the MonTransit app (free) in the "More ..." section bellow or by following this Google Play link http://goo.gl/pCk5mV

You can install this application on the SD card but it is not recommended.

The information comes from the GTFS file provided by the Region of Niagara.
https://niagaraopendata.ca/dataset/niagara-region-transit-gtfs

This application is free and open-source:
https://github.com/mtransitapps/ca-niagara-falls-transit-bus-android

This app is not related with the Region of Niagara and Niagara Falls Transit.
